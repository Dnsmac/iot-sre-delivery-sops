# 目标 JD 画像（深圳 · 25K 区间）

## A 类 — 主攻（每周至少 2 投）

**标题关键词**：IoT 平台运维、技术交付工程师、MQTT/EMQX、物联网实施、云平台交付

| 要求 | 你现状 | 证据计划 |
|------|--------|----------|
| K8s 部署运维 | 生产适配过，原理弱 | W1-2 映射表 + SOP |
| 大规模设备/连接 | 40W 压测负责人 | W3 复盘 md |
| MQTT/物联网协议 | 会用，原理弱 | W4 协议卡片 |
| 中间件 | Redis/ES/MQ/达梦常用 | W6 排障手册 |
| 文档/沟通 | 弱 | 每周口述稿 + W8 录音 |

**标题关键词**：K8s 运维开发、Java 平台、云原生工程师（中小厂）

| 缺口 | 补齐周次 |

## C 类 — 慎投

- 大厂核心 SRE（on-call 年限硬杠）
- 纯 Java 业务开发（重复外包路径）
- 人力外包派遣（除非临时现金流）

## 公司类型（深圳）

| 优先 | 回避 |
|------|------|
| 物联网硬件、模组、车联网、智慧园区 ISV | 纯人力外包池 |
| 有自有平台的产品公司 | 无技术栈的驻场 |
| 20～500 人技术团队可见 | 简历海投无技术面试 |

## 25K 对标（2025～2026 粗算）

- **25K × 13 ≈ 32.5W 年包**（常见深圳结构）
- 你曾 30K ≈ 39W peak → 要 25K 是 **降档回归**，面试话术是「恢复平台岗市价，不是涨价要挟」
- 专升本：主攻 **业务型科技公司**，少碰卡第一学历的头部互联网
